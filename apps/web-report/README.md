# Web Report App

Interactive local report viewer for RepoArena benchmark summaries.

## What It Does

- loads a `summary.json` file generated by RepoArena
- shows run metadata, preflights, and agent results
- lets you switch between agents without scrolling through a long static page
- renders setup, judges, teardown, and diff details in one view

## Usage

```bash
pnpm --filter @repoarena/web-report build
```

Then open `apps/web-report/dist/index.html` in a browser and load a `summary.json` file from `.repoarena/runs/<run-id>/summary.json`.
